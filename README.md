# Vibe coding 你也行：软件开发全景图

AI 时代让编程门槛变得很低。如果你只是想做一个类似贪吃蛇这样的页面，确实很简单，说一句话就可以。

但如果你想做一个真正可发布、可运营、可持续迭代的系统，一句话是不够的。你还是要做很多事情，也要懂一些基本流程和概念，这样和 AI 沟通起来才更高效。

这几个月，我用 AI 开发并推进了这些产品：

- 火火猫 AI 起名，形态：微信小程序  
  小程序码：https://www.huohuomao.com/images/%E5%B0%8F%E7%A8%8B%E5%BA%8F%E7%A0%81.jpg
- AI 人生模拟器游戏，形态：微信小游戏（人生穿越器，国内）、App（LifeLife，海外）  
  App Store：https://apps.apple.com/us/app/lifelife/id6755062821  
  Google Play：https://play.google.com/store/apps/details?id=com.lifelife.app
- 亏了么，形态：iOS App（海外）  
  App Store：https://apps.apple.com/my/app/%E4%BA%8F%E4%BA%86%E4%B9%88/id6758881277
- Agent 相亲平台，形态：网站  
  官网：https://www.agenlace.com
- 语言输入，形态：Mac 端软件（待发布）
- 命令行交互智能增强软件，形态：脚本（待发布）

基本上，相关流程我都亲自走了一遍。

所以我把这些经验总结成这张图，分享给非计算机科班人士，或者毕业后已经把软件技能还给学校的人。

一图胜千言。用图的形式，而且还是可以交互的图，会更直观一些。希望对你有帮助。

## 使用方式

下载仓库里的 `vibe-coding-overview-map.html`，然后直接用浏览器打开即可。

打开后有两种看法：

- 自行拖动、缩放、点击模块探索
- 使用“流程演示模式”，一步一步走完整个流程
